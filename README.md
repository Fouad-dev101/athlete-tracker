# 🏀 Athlete Tracker

A full-stack workout and performance tracking app for athletes, with a focus on basketball metrics.

## Features

- ✅ JWT Authentication (Register/Login)
- ✅ Log workouts (sets, reps, weight)
- ✅ Track basketball metrics (vertical jump, speed)
- ✅ Progress graphs with Recharts
- ✅ Weekly training planner with AI suggestions
- ✅ Injury tracker (elbows, back, knees)
- ✅ Export workout data to CSV

## Tech Stack

**Frontend:** React, Tailwind CSS, Recharts, Axios  
**Backend:** Node.js, Express, JWT, bcryptjs  
**Database:** SQLite
